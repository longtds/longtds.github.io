# 02_Linux · 概述

> Linux 是数据中心和云服务器的事实标准操作系统，掌握 Linux 是一切 SRE/运维工作的根基。

## 一、Linux 发展简史

| 阶段 | 时期 | 里程碑 |
|:---|:---:|:---|
| 内核诞生 | 1991 | Linus Torvalds 发布 0.01 版 |
| 商业化 | 1993-2000 | Slackware/Red Hat/Debian/SUSE 发行版兴起 |
| 企业化 | 2000-2010 | RHEL、CentOS 主导服务器市场 |
| 虚拟化时代 | 2007-2015 | KVM 内置内核、cgroup/namespace |
| 云原生时代 | 2013-至今 | Docker/K8s 推动 Linux 容器化 |
| 沉浸式发行 | 2020-至今 | 不可变镜像、Fedora Silverblue、AIOps |

## 二、主流发行版谱系


## 四、运维核心能力地图

| 维度 | 关注点 |
|:---|:---|
| **系统管理** | 用户/服务/包管理/Systemd |
| **存储** | 文件系统/LVM/RAID 软件层 |
| **调优** | CPU/内存/IO/网络参数 |
| **排查** | 性能工具六字诀 USE/RED |
| **维护** | 备份/补丁/日志/审计 |
| **脚本** | Bash/Awk/Sed 三剑客 |

## 五、本章学习路径

1. **系统管理** → 用户、systemd、包管理
2. **存储与文件系统** → ext4/xfs、LVM、磁盘配额
3. **系统调优** → sysctl、cgroup、ulimit
4. **性能排查** → top/iostat/sar、火焰图、eBPF
5. **系统维护** → 日志/审计/备份/补丁
6. **Shell 脚本** → 自动化运维基础

> 📖 顺序学习，每章配套一个真实运维场景案例。
