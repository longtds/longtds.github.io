# CTF 实战

## 靶机平台

| 平台 | 地址 | 特点 |
|:---|:---|:---|
| VulnHub | vulnhub.com | 离线靶机 |
| HackTheBox | hackthebox.com | 在线 CTF |
| TryHackMe | tryhackme.com | 新手友好 |
| PentesterLab | pentesterlab.com | Web 专项 |

## 常见题型

- Web：SQL 注入/XSS/文件上传/SSRF
- PWN：二进制漏洞利用
- Reverse：逆向工程
- Crypto：密码学
- Misc：杂项（流量分析/隐写）
