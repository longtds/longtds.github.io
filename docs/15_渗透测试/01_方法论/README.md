# 渗透测试方法论

## PTES 标准

PTES（Penetration Testing Execution Standard）定义 7 个阶段：

| 阶段 | 内容 |
|:---|:---|
| 1. 前期交互 | 确定范围、目标、规则 |
| 2. 情报收集 | 信息收集 |
| 3. 威胁建模 | 分析攻击路径 |
| 4. 漏洞分析 | 漏洞检测 |
| 5. 漏洞利用 | 获取权限 |
| 6. 后渗透 | 横向移动/提权 |
| 7. 报告 | 输出发现 |

## 渗透测试报告模板

```markdown
# 渗透测试报告

## 概述
- 测试范围
- 测试时间
- 测试方法

## 发现汇总
| 编号 | 漏洞 | 严重等级 | 状态 |
|:---:|:---|:---:|:---:|
| 1 | XSS | 高 | 未修复 |

## 详细发现
### 1. XSS 跨站脚本攻击
- 位置：/search?q=
- 影响：窃取用户 Cookie
- 复现：<script>alert(1)</script>
- 修复：输出编码/过滤 HTML
```
