# 网络设备与运维

## 交换机

| 层级 | 角色 | 功能 |
|:---|:---|:---|
| 接入层 | 连接服务器 | VLAN 划分、端口安全 |
| 汇聚层 | 汇聚接入层 | 路由、ACL、QoS |
| 核心层 | 高速转发 | 三层交换、BGP |

## 路由器

- 静态路由：手动配置，适合小规模
- 动态路由：OSPF/BGP，自动学习
- 策略路由：基于源 IP/应用的选路

## 防火墙

- 状态检测：维护连接状态表
- NAT：源 NAT/目标 NAT/端口映射
- DMZ：隔离内外网

## 负载均衡

| 层级 | 原理 | 工具 |
|:---|:---|:---|
| 四层 | TCP/UDP 转发 | LVS/HAProxy |
| 七层 | HTTP 内容分发 | Nginx/HAProxy |
