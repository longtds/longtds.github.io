# 传输层协议

## TCP

TCP 是面向连接的可靠传输协议。

### 三次握手

```
Client                  Server
   │                       │
   ├── SYN ──────────────► │
   │                       │
   │ ◄── SYN+ACK ──────── │
   │                       │
   ├── ACK ──────────────► │
   │                       │
   └── 连接建立，开始传输 ──┘
```

### 四次挥手

```
Client                  Server
   │                       │
   ├── FIN ──────────────► │
   │ ◄── ACK ──────────── │
   │ ◄── FIN ──────────── │
   ├── ACK ──────────────► │
   │                       │
```

### 拥塞控制算法

| 算法 | 特点 | 场景 |
|:---|:---|:---|
| CUBIC | Linux 默认 | 通用 |
| BBR | Google 开发，带宽利用率高 | 长肥网络 |
| BBRv3 | 改进版 | 生产环境 |

## UDP

UDP 是无连接的不可靠传输协议，适用于实时通信场景。

| 特性 | TCP | UDP |
|:---|:---|:---|
| 连接 | 面向连接 | 无连接 |
| 可靠性 | 可靠（重传+ACK） | 不可靠 |
| 有序性 | 保证 | 不保证 |
| 延迟 | 较高 | 低 |
| 场景 | Web/DB/邮件 | 视频/DNS/游戏 |
