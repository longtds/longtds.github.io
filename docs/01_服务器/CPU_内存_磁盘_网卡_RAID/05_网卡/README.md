# 网卡

## 概述

网卡是服务器与网络的连接接口。从千兆到 100G，从标准网卡到智能网卡，选择日益多样化。

## 速度标准

| 标准 | 速率 | 适用场景 |
|:---|:---:|:---|
| 1GbE | 1Gbps | 管理网络 |
| 10GbE | 10Gbps | 通用业务 |
| 25GbE | 25Gbps | 高性能业务 |
| 100GbE | 100Gbps | AI/存储网络 |
| 200/400GbE | 200-400Gbps | 超大规模 |

## 技术特性

- **SR-IOV**：单根 I/O 虚拟化，虚拟机直通网卡
- **DPDK**：数据平面开发套件，绕过内核协议栈
- **RDMA**：远程直接内存访问，低延迟通信
- **网卡多队列**：多 CPU 核心并行处理网络数据
