# 内存

## 概述

服务器内存是临时数据存储的关键组件，直接影响应用性能和并发处理能力。

## 内存类型

| 类型 | 说明 | 典型容量 |
|:---|:---|:---:|
| DDR4 | 上一代标准，性价比高 | 16GB/32GB/64GB |
| DDR5 | 新一代标准，带宽更高 | 32GB/64GB/128GB |
| ECC | 纠错内存，服务器标配 | - |
| RDIMM | 注册内存，降低信号负载 | - |
| LRDIMM | 低负载内存，支持更大容量 | - |

## 配置原则

- 多通道配置（每路 CPU 通常 8 通道）
- 不同容量/规格的内存不要混插
- 优先填满离 CPU 最近的插槽
- 所有内存以最低频率运行

## 常见问题

- 内存故障定位（BMC 日志/MEMTEST 测试）
- 容量规划（根据应用类型估算）
- ECC 纠错日志解读
