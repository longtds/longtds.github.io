# RAID

## 概述

RAID（Redundant Array of Independent Disks）通过将多块磁盘组合为一个逻辑卷，提供数据冗余和/或性能提升。

## 常用 RAID 级别

| 级别 | 最少磁盘 | 容量利用率 | 特点 | 场景 |
|:---|:---:|:---:|:---|:---|
| RAID 0 | 2 | 100% | 条带化，无冗余 | 临时数据 |
| RAID 1 | 2 | 50% | 镜像，高可靠 | 系统盘 |
| RAID 5 | 3 | (N-1)/N | 奇偶校验 | 通用数据 |
| RAID 6 | 4 | (N-2)/N | 双重校验，可坏2盘 | 大容量存储 |
| RAID 10 | 4 | 50% | 条带+镜像 | 数据库 |

## 配置要点

- **缓存策略**：Write Back 提升写性能（需电池保护），Write Through 更安全
- **热备盘**：配置全局或专用热备盘，自动替换故障盘
- **条带大小**：大块（64KB+）适合顺序读写，小块（16-32KB）适合随机读写
