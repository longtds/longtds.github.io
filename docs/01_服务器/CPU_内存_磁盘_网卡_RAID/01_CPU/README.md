# CPU

## 概述

CPU 是服务器的计算核心。x86 架构是目前服务器市场的主流，Intel 和 AMD 是两大主要供应商。

## 核心概念

- **物理核**：实际存在的硬件核心
- **逻辑核**：通过超线程（Hyper-Threading）技术，一个物理核模拟为两个逻辑核
- **NUMA**（Non-Uniform Memory Access）：CPU 访问本地内存和远端内存的延迟不同
- **缓存层级**：L1/L2/L3 Cache 的容量与延迟

## 关键指标

| 指标 | 说明 | 重要性 |
|:---|:---|:---:|
| 主频 | CPU 运行频率（GHz） | 高 |
| 核心数 | 物理核心数量 | 高 |
| 缓存 | L3 缓存容量 | 中 |
| TDP | 热设计功耗 | 中 |
| 指令集 | AVX-512/AVX2/AMX | AI 算力场景 |
