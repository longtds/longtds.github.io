# 01_服务器 · 概述

> 从大型机到 AI 服务器，从单核到 Chiplet，服务器是支撑整个 IT 体系的物理基石。

## 一、服务器发展简史

| 阶段 | 时期 | 代表 | 特征 |
|:---|:---:|:---|:---|
| 大型机时代 | 1960s | IBM System/360 | 集中式计算，专用硬件 |
| 小型机时代 | 1970s-1990s | DEC VAX、IBM AS/400 | UNIX 工作站，私有架构 |
| x86 服务器时代 | 1990s-2010s | Intel Xeon、AMD Opteron | 通用架构，标准化 |
| 虚拟化时代 | 2005-2015 | VMware ESX、KVM | 一虚多，资源池化 |
| 云服务器时代 | 2010-至今 | AWS EC2、阿里云 ECS | 弹性计算，按需付费 |
| AI 服务器时代 | 2020-至今 | DGX H100、HGX B200 | GPU 异构，超大显存互联 |

## 二、服务器体系架构演进

### 2.1 从南北桥到 PCIe 直连

```
传统架构（2005-） 现代架构（2015+）

CPU ─ 北桥 ─ 内存 CPU ─── 内存（DDR5）
 │ │
 └── 南桥 ─ PCIe/SATA ├── PCIe 5.0 ──── GPU/NVMe
 │
 └── CXL ────────── 内存池
```

### 2.2 NUMA 架构

多路 CPU 服务器中，每颗 CPU 有本地内存，跨 CPU 访问内存有性能损失。运维要关注 NUMA 亲和性绑定。

```bash
numactl --hardware # 查看 NUMA 拓扑
numactl --cpunodebind=0 --membind=0 ./app # 绑定到 Node 0
```

### 2.3 Chiplet（小芯片）

将一颗大芯片拆成多个 Chiplet 用 EMIB/CoWoS 互联，良率↑成本↓。AMD EPYC、Intel Xeon 都已采用。

## 三、服务器形态对比

| 形态 | 优势 | 劣势 | 典型场景 |
|:---|:---|:---|:---|
| **塔式** | 安静、扩展性好 | 占地大 | 中小企业、边缘 |
| **机架式** (1U/2U/4U) | 密度高、标准化 | 散热挑战 | 数据中心主流 |
| **刀片** | 极高密度、共享供电 | 厂商锁定 | 大型企业 |
| **整机柜** | 出厂即部署，液冷友好 | 定制化 | 超大规模数据中心 |
| **AI 服务器** | GPU+NVLink 互联 | 功耗 5-10kW | 训练/推理 |
