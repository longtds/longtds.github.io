# GPU 互联

## 概述

多 GPU 互联是大规模 AI 训练和推理的关键技术。GPU 间的通信带宽直接影响多卡扩展效率。

## 互联技术对比

| 技术 | 带宽（单向） | 延迟 | 拓扑 | 场景 |
|:---|:---:|:---:|:---|:---|
| PCIe 4.0 x16 | 32GB/s | ~1μs | 树形 | 通用 |
| PCIe 5.0 x16 | 64GB/s | ~0.5μs | 树形 | 新一代通用 |
| NVLink 3.0 | 600GB/s | ~100ns | 全互联 | H100 互联 |
| NVSwitch | 900GB/s+ | ~100ns | 交换网络 | 大规模集群 |

## NCCL 与拓扑

NCCL（NVIDIA Collective Communications Library）是多 GPU 通信库，其性能高度依赖 GPU 互联拓扑。
