# 容器 vs 虚拟机

## 核心区别

| 维度 | 虚拟机 | 容器 |
|:---|:---|:---|
| 隔离级别 | OS 级别 | 进程级别 |
| Guest OS | 每个 VM 独立 OS | 共享宿主机内核 |
| 启动时间 | 分钟级 | 秒级 |
| 资源占用 | GB 级 | MB 级 |
| 性能损耗 | 5-15% | 近乎零 |
| 迁移性 | 虚拟机格式绑定 | 镜像标准 OCI |

## 适用场景

| 场景 | 推荐 | 原因 |
|:---|:---|:---|
| 传统单体应用 | VM | 兼容性好 |
| 微服务 | 容器 | 轻量/弹性 |
| 安全敏感 | VM | 更强隔离 |
| 开发测试 | 容器 | 快速迭代 |
| GPU 计算 | 容器 | 低性能损耗 |

## 趋势：VM on K8s

KubeVirt 项目实现了在 K8s 中运行虚拟机，统一管理容器和 VM。
