# 告警管理

## 告警降噪

| 问题 | 降噪策略 |
|:---|:---|
| 重复告警 | 告警聚合（相同 Pod 同类告警合并） |
| 抖动告警 | 持续 N 分钟才触发（for: 5m） |
| 已知告警 | 维护静默规则（已知维护窗口） |
| 关联告警 | 事件关联（同一 Pod CPU+内存+网络同时告警 = root cause 一条） |

## 告警分级

| 级别 | 响应方式 | 示例 |
|:---|:---|:---|
| P0（严重） | 立即电话/短信 | 集群宕机、核心服务不可用 |
| P1（高） | 15min 内响应 | 部分服务故障 |
| P2（中） | 工作时间处理 | 单节点异常 |
| P3（低） | 记录，周会回顾 | 磁盘使用率 > 80% |
