## 📝 使用说明

- 章节按**学习路径与依赖关系**重新编排（基础设施→容器→工程→数据→AI→安全→排查）
- 每章按 **基础原理 → 部署实践 → 运维调优 → 故障排查** 深度递进
- `99_发展与展望` 是每章的未来视角
- `16_故障排查` 使用统一模板，每次修故障顺手记
- 代码块中的命令可直接复制执行，注意替换占位参数
