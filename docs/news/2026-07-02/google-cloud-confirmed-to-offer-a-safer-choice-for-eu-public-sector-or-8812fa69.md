在 Google Cloud，我们致力于为全球公共部门组织提供高度灵活、可扩展且按照市场领先的数据保护、主权和安全标准构建的云技术。我们知道，对于欧盟的公共部门组织来说，对数据保护的信心不仅是一种偏好，而且是一个先决条件。

今天，我们很高兴地宣布一个重要的里程碑，强化了对 Google Cloud 的这一承诺。

#### 荷兰政府 DPIA 确认 

Google Cloud 拥有强大的隐私基础

我们与荷兰政府的战略供应商管理机构 [SLM Rijk](https://www.digitaleoverheid.nl/overzicht-van-alle-onderwerpen/slm-rijk/slm-mga/) 成功合作，后者完成了对 Google Cloud 的严格数据保护影响评估 (DPIA)。

此次合作证实了 Google Cloud 坚定致力于加强荷兰公共部门对其隐私状况的信任。鉴于 DPIA 期间提出的所有关键点均已成功解决（请参阅 SLM Rijk 的摘要[此处](https://open.overheid.nl/details/3ef89ab7-ccaf-4753-8335-9f226eaa9c24)），并且他们的 DPIA 得出的结论是，实施建议的措施时不存在已知的高数据保护风险，因此从隐私评估的角度来看，荷兰中央公共部门现在可以正式使用 Google Cloud，并具有明确的路径。

因此，我们鼓励荷兰中央公共部门的潜在客户和客户与我们互动，了解有关 Google Cloud 的更多信息。

更广泛地说，我们相信这一成果为荷兰及其他地区的公共部门组织自信地评估和采用 Google Cloud、安全地实现现代化和数字化转型奠定了坚实的基础。这一成就建立在我们在该地区的良好记录之上，包括成功完成 [Google Workspace 上的荷兰语 DPIA](https://workspace.google.com/blog/identity-and-security/eu-public-sector-dutch-approval-and-new-capabilities?e=48754805)。之前的成功证实了荷兰公共部门和教育机构可以安全使用 Workspace。这些评估共同证明了 Google 持续致力于帮助公共部门组织满足其隐私、安全和合规性要求，同时受益于 Google Cloud 的创新和可扩展性。

#### 为所有客户的合规之旅提供持续支持

我们欢迎有助于加强信任、透明度和问责制的独立评估。荷兰政府的 DPIA 流程是公共机构、独立专家和云提供商之间建设性合作以推进隐私保护的重要范例。

Google Cloud 始终致力于帮助客户履行合规义务，同时提供安全、透明且注重隐私的云服务。

我们将继续投资于隐私增强技术、透明度举措和客户控制，以支持欧洲和世界各地的组织。

我们亲身了解，执行 DPIA 可能是一项复杂的任务，因此我们仍然坚定地致力于利用我们综合性的 [DPIA 云资源中心](https://cloud.google.com/privacy/data-protection-impact-assessment) 中的资源帮助客户驾驭 DPIA。
