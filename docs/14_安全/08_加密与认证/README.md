# 加密与认证

## 对称加密

| 算法 | 密钥长度 | 安全性 |
|:---|:---:|:---:|
| AES-256 | 256bit | 高 |
| SM4 | 128bit | 国密标准 |

## 非对称加密

| 算法 | 密钥长度 | 场景 |
|:---|:---:|:---|
| RSA | 2048/4096bit | 签名/加密 |
| ECC | 256bit | 移动端 |
| SM2 | 256bit | 国密 |

## TLS 握手

1. ClientHello（支持的密码套件）
2. ServerHello（选择套件+证书）
3. 证书验证
4. 密钥交换
5. 加密通信开始
