# 大数据技术演进

## Hadoop 时代（2006-2015）

- HDFS：分布式文件系统
- MapReduce：批处理计算模型
- Hive：SQL on Hadoop
- HBase：NoSQL 列式数据库

## Spark 时代（2015-2020）

- 内存计算，比 MapReduce 快 10-100 倍
- RDD/DataFrame/Dataset 统一编程模型
- Spark SQL / Streaming / MLlib 一体化

## 云原生数据湖时代（2020-至今）

- Spark on K8s，不再依赖 YARN
- 对象存储（S3/MinIO）替代 HDFS
- Delta Lake / Hudi / Iceberg 提供 ACID 语义
- 计算存储分离，独立扩缩容
