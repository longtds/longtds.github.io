# 云平台选型

## OpenStack vs Kubernetes

| 维度 | OpenStack | Kubernetes |
|:---|:---|:---|
| 抽象层级 | 虚拟化（提供 VM） | 容器编排（提供 Pod） |
| 资源粒度 | VM（GB 级） | 容器（MB 级） |
| 启动速度 | 分钟级 | 秒级 |
| 运维复杂度 | 高（10+ 组件） | 中 |
| 生态系统 | 成熟稳定 | 快速增长 |

## 迁移策略

- 容器化改造（12-Factor App）
- 先边缘服务后核心服务
- KubeVirt 作为过渡方案（在 K8s 中运行 VM）

## 混合云策略

- 敏感数据在私有云，弹性算力走公有云
- 统一网络（VPN/专线）
- 统一认证（SSO）
