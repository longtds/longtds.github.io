# 07_Kubernetes

## 概述

Kubernetes 是当前容器编排的事实标准。本章从基础概念到集群运维，覆盖 K8s 全栈知识体系。

## 目录

- 基础概念 — 架构/核心对象/工作负载
- 集群部署 — kubeadm/Kubespray/HA/升级
- 网络 — CNI/Cilium/Calico/Ingress
- 存储 — CSI/Rook-Ceph/MinIO
- 调度与资源管理 — QoS/调度策略/HPA/GPU
- Operator 开发 — CRD/Controller/Reconcile
- 安全 — RBAC/PSP/OPA/Kubescape
- CI/CD — Jenkins/ArgoCD/Harbor/Keptn
- 可观测性 — Prometheus/Grafana/OpenObserve
- 集群运维 — ETCD/备份/证书
- 生态工具 — Helm/K9s/KubeSphere
